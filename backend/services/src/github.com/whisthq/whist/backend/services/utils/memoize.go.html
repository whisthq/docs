     4  // error)) into another function of the same type.  The function returned from
     5  // MemoizeStringWithError does not cache the results from a failed call (i.e.
     6  // non-nil error). It only caches the return value from a successful call to
     7  // the argument function `f`. However, once the argument function `f` is called
     8  // successfully once, all future calls to the returned function will return the
     9  // cached result of the successful call, and a nil error.
