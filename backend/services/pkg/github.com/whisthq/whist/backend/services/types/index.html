Package types simply contains some useful types for the `mandelbox` package. We define this package separately so that we can safely pass these types around to other packages that `mandelbox` itself might depend on.

type MandelboxID ¶

A MandelboxID is a random string that the webserver creates for each mandelbox. We need some sort of identifier for each mandelbox, and we need it _before_ Docker gives us back the runtime Docker ID for the mandelbox. 

type MandelboxID uuid.UUID
