type PrivateKey ¶

PrivateKey is an AES key used for connecting the protocol client to the protocol server running inside a mandelbox. Its purpose is to prevent other users from connecting to a mandelbox that is not assigned to them. 

type PrivateKey string

type SessionID ¶

SessionID can be a random hash or a timestamp, depending if it's a server-side session id, or a client-side session id. A server-side session id is created by the host service when starting a waiting mandelbox, and is represented by a random hash. A client-side session id is created by the frontend and is written when a mandelbox gets assigned to a user, it's represented by a Unix timestamp. 

type SessionID string

UserID is the id assigned to a user by the authentication provider (Auth0).
