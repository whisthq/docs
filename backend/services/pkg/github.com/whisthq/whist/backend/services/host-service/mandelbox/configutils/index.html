func EncryptAES256GCM ¶ 

func EncryptAES256GCM(password string, data []byte) ([]byte, error)

EncryptAES256GCM takes a password and plaintext data and encrypts the data using GCM. AES-GCM is an AEAD (authenticated encryption with associated data) method that no only provides data confidentiality but also a way to verify authenticity/integrity of the data. GCM is also highly performant and fully-parallelizable.

See: https://en.wikipedia.org/wiki/Galois/Counter_Mode Based on example code from: https://golang.org/src/crypto/cipher/example_test.go
