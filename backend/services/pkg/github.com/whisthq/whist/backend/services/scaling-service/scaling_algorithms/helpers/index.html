func ComputeExpectedMandelboxCapacity(imageID string, activeInstances []subscriptions.Instance, startingInstances []subscriptions.Instance) int64

ComputeExpectedMandelboxCapacity is a helper function to compute the expected mandelbox capacity. Expected capacity is the number of free mandelboxes available on instances with active status and in starting instances.

func ComputeRealMandelboxCapacity(imageID string, activeInstances []subscriptions.Instance) int64

ComputeRealMandelboxCapacity is a helper function to compute the real mandelbox capacity. Real capcity is the number of free mandelboxes available on instances with active status.

func SanitizeEmail ¶ 

func SanitizeEmail(email string) (string, error)

SanitizeEmail tries to match an email to a general email regex and if it fails, returns an empty string. This is done because the email can be spoofed from the frontend.
