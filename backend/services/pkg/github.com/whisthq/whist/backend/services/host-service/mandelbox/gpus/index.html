MaxMandelboxesPerGPU represents the maximum of mandelboxes we can assign to each GPU and still maintain acceptable performance. Note that we are assuming that all GPUs on a mandelbox are uniform (and that we are using only one instance type). 

const MaxMandelboxesPerGPU = 3

func Free ¶ 

func Free(index Index, mandelboxID types.MandelboxID) error

Free decrements the number of mandelboxes we consider assigned to a given GPU indexed by `index`. The Mandelbox id is used to determine if the mandelbox was assigned to a given GPU.

type GPU ¶

GPU holds the state for any given GPU. It contains an array of mandelbox ids, which is used to ensure we only free assigned resources.

Index is essentially a handle to a GPU on the system. It is the index of a particular GPU in the output of `nvidia-smi`.
