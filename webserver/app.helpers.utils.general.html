app.helpers.utils.general.logs.fractal_task_prerun(task_id, task, **kwargs)¶

See https://docs.celeryproject.org/en/stable/userguide/signals.html#task-prerun. This is run before any celery task is executed. We use this to print out the args and kwargs of every celery task so its input state is clear.
