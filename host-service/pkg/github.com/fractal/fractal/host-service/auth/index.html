Package auth provides functions for validating JWTs sent by the client app.

Currently, it has been tested JWTs generated with our Auth0 configuration. It should work with other JWTs too, provided that they are signed with the RS256 algorithm.

type FractalClaims ¶

FractalClaims is a struct type that models the claims that must be present in an Auth0-issued Fractal access token. 

type FractalClaims struct {
    jwt.StandardClaims

    // Audience stores the value of the access token's "aud" claim. Inside the
    // token's payload, the value of the "aud" claim can be either a JSON
    // string or list of strings. We implement custom deserialization on the
    // Audience type to handle both of these cases.
    Audience Audience `json:"aud"`

    // Scopes stores the value of the access token's "scope" claim. The value
    // of the "scope" claim is a string of one or more space-separated words.
    // The *Scopes type implements the encoding.TextUnmarshaler interface such
    // that the string of space-separated words is deserialized into a string
    // slice.
    Scopes Scopes `json:"scope"`
}

func Verify ¶ 

func Verify(tokenString string) (*FractalClaims, error)

Verify parses an raw access token string, verifies the token's signature, ensures that it is valid at the current moment in time, and checks that it was issued by the proper issuer for the proper audience. It returns a pointer to a FractalClaims type containing the values of its claims if all checks are successful.

func (*FractalClaims) VerifyAudience ¶ 

func (claims *FractalClaims) VerifyAudience(cmp string, req bool) bool

VerifyAudience compares the "aud" claim against cmp. If req is false, this method will return true if the value of the "aud" claim matches cmp or is unset.

func (*FractalClaims) VerifyScope ¶ 

func (claims *FractalClaims) VerifyScope(scope string) bool

VerifyScope returns true if claims.Scopes contains the requested scope and false otherwise. This function's name and type signature is inspired by those of the Verify* methods defined on jwt.StandardClaims.
