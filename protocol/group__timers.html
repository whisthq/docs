Detailed Description

Whist API for timers.

This provides a way to measure how long a section of code takes or how much real time has elapsed since some event.

Example:

WhistTimer timer;
start_timer(&timer);
foo();
printf("foo() took %f seconds to run.", get_timer(&timer));

◆ get_timestamp_sec()

double get_timestamp_sec ( void  )

Get a comparable timestamp in sec.

Returns
Timestamp in sec.
Note
The point of this function is to get a comparable timestamp, the absolute value is not important. But for a well-defined behavior: The first call of this function returns zero, following calls run the time past in sec since first call. This function is thread-safe after the first call finishes. The first call is expected to be done inside whist_init_subsystems().
