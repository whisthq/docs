Detailed Description

Contains utilities to capture and handle D-Bus messages.

Usage

In the main function of the server, one can create a thread and destroy it later.

DBusHandlerContext* dbus_handler = whist_create_dbus_handler(Wserver_state); execute main loop of server whist_destroy_dbus_handler(dbus_handler);
