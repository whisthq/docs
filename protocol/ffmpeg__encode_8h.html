Detailed Description

Copyright (c) 2021-2022 Whist Technologies, Inc.

This file contains the code to create and destroy FFmpeg encoders and use

them to encode captured screens.

Usage

Video is encoded to H264 via either a hardware encoder (currently, we use NVidia GPUs, so we use NVENC) or a software encoder if hardware encoding fails. H265 is also supported but not currently used. For encoders, create an H264 encoder via create_ffmpeg_encoder, and use it to encode frames via ffmpeg_encoder_send_frame. Retrieve encoded packets using ffmpeg_encoder_receive_packet. When finished, destroy the encoder using destroy_ffmpeg_encoder.

typedef struct FFmpegEncoder FFmpegEncoder

Struct for handling ffmpeg encoding of video frames.

If software encoding, the codec and context determine the properties of the output frames, and scaling is done using the filter_graph. Frames encoded on the GPU are stored in hw_frame, while frames encoded on the CPU are stored in sw_frame.
