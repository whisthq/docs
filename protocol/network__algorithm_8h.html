Detailed Description

Copyright (c) 2021-2022 Whist Technologies, Inc.

This file contains the client's network algorithm code. It takes in any network statistics,

and returns a network settings decision based on those statistics.

Usage

The client should periodically call calculate_new_bitrate with any new network statistics, such as throughput or number of nacks. This will return a new network settings as needed
