Detailed Description

Copyright (c) 2021-2022 Whist Technologies, Inc.

@brief This file defines the methods required to implement an input driver

Usage

Create an input device by calling CreateInputDevice(). Now, you may use this device to query keyboard state, or to emit mouse button, mouse motion, scroll, and keyboard events. These methods are used, for example, in input.c, to build WhistClientMessage handlers and to generate keyboard input from an arbitrary string.
