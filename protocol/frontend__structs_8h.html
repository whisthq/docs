Enumerator
FRONTEND_EVENT_UNHANDLED 
FRONTEND_EVENT_RESIZE 

Resize event.

The output window has been resized, so anything which draws to the window should adjust to use the new size.

FRONTEND_EVENT_VISIBILITY 

Visibility change event.

When not visible, nothing needs to be drawn to the window.

FRONTEND_EVENT_AUDIO_UPDATE 

Audio device update event.

An audio device has been added or removed, so if audio is playing then the device state will need to be refreshed.

FRONTEND_EVENT_KEYPRESS 

Key press event.

A key have been either pressed or released.

FRONTEND_EVENT_MOUSE_MOTION 

Mouse motion event.

FRONTEND_EVENT_MOUSE_BUTTON 

Mouse button event.

A mouse button was either pressed or released.

FRONTEND_EVENT_MOUSE_WHEEL 

Mouse wheel event.

FRONTEND_EVENT_MOUSE_LEAVE 

Mouse leave event.

The mouse has left the output window.

FRONTEND_EVENT_GESTURE 

Gesture event.

FRONTEND_EVENT_FILE_DRAG 

File drag event.

A file is being dragged over the window.

FRONTEND_EVENT_FILE_DROP 

File drop event.

FRONTEND_EVENT_OPEN_URL 

URL open event.

FRONTEND_EVENT_STARTUP_PARAMETER 

Startup parameter event.

FRONTEND_EVENT_QUIT 

Quit event.

The user or OS indicated that the application should quit.

FRONTEND_EVENT_INTERRUPT 

Interrupt event.

Used to interrupt the main thread while it is waiting for events when there is some external work for it to do. This is triggered by calling the whist_frontend_interrupt() function from another thread.
