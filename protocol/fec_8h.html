int  index,
void *  buffer,
int  buffer_size 
)

Registers a buffer into the decoder.

Parameters
fec_decoderThe FEC decoder to use
indexThe index of the buffer that's being registered. Must be between 0 and num_real_buffers + num_fec_buffers - 1
bufferThe buffer to register
buffer_sizeThe size of the buffer that's being registered
Note
All buffers passed into this function should be some subset of the buffers returned by fec_get_encoded_buffers, with the index indexing into the arrays returned by that function.

The data pointed to by the buffer being passed in, must be held alive for as long as the fec_decoder is alive.

Gets the encoded buffers.

Parameters

buffersThe array of void* buffers for the FEC encoded packets
buffer_sizesThe array of int buffer sizes of the FEC encoded packets
Note
