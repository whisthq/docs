Detailed Description

Copyright Fractal Computers, Inc. 2020

@brief This file defines the cursor types, functions, init and get.

Usage

Use InitCursor to load the appropriate cursor images for a specific OS, and then GetCurrentCursor to retrieve what the cursor shold be on the OS (drag-window, arrow, etc.).
