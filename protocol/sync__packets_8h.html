@brief This file contains code for high-level communication with the server

Usage

Use init_packet_synchronizers() to spin up threads for handling UDP messages (video, audio, etc.) and TCP messages (clipboard, files, etc.). Use destroy_packet_synchronizers() to shut them down and wait for them to finish.
