Detailed Description

@brief This file contains code for high-level communication with the server

Usage

Use multithreaded_sync_udp_packets to send and receive UDP messages to and from the server, such as audio and video packets. Use multithreaded_sync_tcp_packets to send and receive TCP messages, mostly clipboard packets.
