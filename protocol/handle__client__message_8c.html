int handle_client_message ( FractalClientMessage *  fcmsg)

Handles message received by the server from a client, on the server-side.

Replays keyboard and mouse input, replies to pings, or performs whatever side-effects appropriate in handling message from client. Logs errors.

Parameters
fcmsgMessage from client
Returns
Returns -1 on failure, 0 on success
