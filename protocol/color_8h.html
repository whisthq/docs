Detailed Description

This file contains functions relating to colors, colorspace conversion, and implementations of algorithms and heuristics related to color and UX.

Copyright 2021 Fractal Computers, Inc., dba Whist

Usage

The conversion functions take an initialized XXXColor struct and return a YYYColor struct. Meanwhile, other functions will appropriately parse the struct and return a useful computation from the color.

Testing

The functionality provided by this file is tested in test/ClientTest.cpp, in FractalColorTest.

◆ FractalYUVColor

typedef struct FractalYUVColor FractalYUVColor

YUV Color.

Y', U, and V are in [0,255], and correspond to the values in YUV420p data. Y' is luma, U is blue-projected chrominance, and V is red-projected chrominance.
