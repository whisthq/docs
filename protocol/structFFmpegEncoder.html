Detailed Description

Struct for handling ffmpeg encoding of video frames. If software encoding, the codec and context determine the properties of the output frames, and scaling is done using the filter_graph. Frames encoded on the GPU are stored in hw_frame, while frames encoded on the CPU are stored in sw_frame.
