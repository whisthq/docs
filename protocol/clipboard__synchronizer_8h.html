Detailed Description

Copyright (c) 2021-2022 Whist Technologies, Inc.

This file contains code meant, to be used on the clientside, that will

assist in synchronizing the client-server clipboard.

Usage

init_clipboard_synchronizer(true);

ClipboardData received_clipboard_chunk;

Will push this chunk onto our clipboard push_clipboard_chunk(&received_clipboard_chunk);

ClipboardData* our_chunk_to_send = pull_clipboard_chunk();

if (our_chunk_to_send) { We have a new clipboard chunk, this should be sent to the server Send(our_chunk_to_send); } else { There is no new clipboard }

destroy_clipboard_synchronizer();
