Detailed Description

This file contains custom memory handling for Fractal.

Copyright Fractal Computers, Inc. 2020

Usage

clipboard = allocate_region(sizeof(ClipboardData) + cb->size); memcpy(clipboard, cb, sizeof(ClipboardData) + cb->size); deallocate_region(clipboard);
