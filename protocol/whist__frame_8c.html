◆ get_total_frame_size()

int get_total_frame_size ( VideoFrame *  frame)

Get the total VideoFrame size, including all of the data embedded in the VideoFrame's buffer.

Even if the VideoFrame* is being stored in a much larger buffer, this function returns only the number of bytes needed for the data inside the VideoFrame* to be read correctly. I.e., these are the only bytes that need to be sent over for example a network connection.

Returns
The number of bytes that the frame uses up.
