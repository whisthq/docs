Detailed Description

Copyright (c) 2021-2022 Whist Technologies, Inc.

This file contains the network interface code for UDP protocol

Usage

To create the context: udp_context = create_udp_network_context(...);

To send a packet: udp_context->send_packet(...);

To read a packet: udp_context->read_packet(...);

To free a packet: udp_context->free_packet(...);

Creates a udp network context and initializes a UDP connection between a server and a client.

Parameters
contextThe socket context that will be initialized
destinationThe server IP address to connect to. Passing NULL will wait for another client to connect to the socket
portThe port to connect to. It will be a virtual port.
recvfrom_timeout_sThe timeout that the socketcontext will use after being initialized
connection_timeout_msThe timeout that will be used when attempting to connect. The handshake sends a few packets back and forth, so the upper bound of how long CreateUDPSocketContext will take is
using_stunTrue/false for whether or not to use the STUN server for this context some small constant times connection_timeout_ms
binary_aes_private_keyThe AES private key used to encrypt the socket communication
Returns
The UDP network context on success, NULL on failure

WhistPacketType  type,
int  max_frame_size,
int  num_buffers 
)

Registers a nack buffer, so that future nacks can be handled.

It will be able to respond to nacks from the most recent buffer_size ID's that have been send via send_packet NOTE: This function is not thread-safe on SocketContext

Parameters
contextThe SocketContext that will have a nack buffer
typeThe WhistPacketType that this nack buffer will be used for
max_frame_sizeThe largest frame that can be saved in the nack buffer
num_buffersThe number of buffers that will be stored in the nack buffer
