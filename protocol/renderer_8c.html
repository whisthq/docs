◆ renderer_try_render()

void renderer_try_render ( WhistRenderer *  renderer)

Render the video/audio frames (If any are available to render)

Parameters
rendererThe video context that wants to render a frame
Note
This function is thread-safe, and may be called in a way that overlaps other functions that use the whist renderer.
This function will potentially take a long time (4-8ms)
This function is optional to call, but it's provided so that rendering work can be done on a thread that already exists. If you fail to call this function, another thread will be used automatically, increasing CPU usage.
