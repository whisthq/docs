◆ sdl_update_pending_tasks()

void sdl_update_pending_tasks ( )

The above functions may be expensive, and thus may have to be queued up. This function executes all of the currently internally queued actions.

Note
This function must be called by the same thread that originally called init_sdl. This will also render out any pending framebuffer, thereby setting sdl_render_pending to false.
