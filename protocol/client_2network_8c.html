Detailed Description

Copyright (c) 2021-2022 Whist Technologies, Inc.

This file contains client-specific wrappers to low-level network

functions.

Usage

connect_to_server, close_connections, and send_server_quit_messages are used to start and end connections to the Whist server. To connect, call connect_to_server. To disconnect, send_server_quit_messages and then close_connections.

To communicate with the server, use send_wcmsg to send Whist messages to the server. Large wcmsg's (e.g. clipboard messages) are sent over TCP; otherwise, messages are sent over UDP.
