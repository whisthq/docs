Detailed Description

Copyright (c) 2021-2022 Whist Technologies, Inc.

This file contains all code that interacts directly with processing

audio packets on the client.

Usage

initAudio() must be called first before receiving any audio packets. updateAudio() gets called immediately after to update the client to the server's audio format.

Parameters

Note
