◆ rs_wrapper_create()

RSWrapper* rs_wrapper_create ( int  num_real_buffers,
int  num_total_buffers 
)

Create an RS wrapper, the rs_wapper provide an interface very similar to the lugi's RS lib but don't have the limitation of num_total_buffers<=256.

when num_total_buffers is too large, the orginal data are partition into groups, encoding and decoding are done group-wise.

Parameters
num_real_buffersnum of original buffers
num_total_buffersnum of buffers in total
Returns
The RS wrapper
