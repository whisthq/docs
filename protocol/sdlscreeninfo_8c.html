Detailed Description

This file contains SDL screen info code, used to retrive information.

Copyright 2021 Fractal Computers, Inc., dba Whist

about SDL Windows.

Usage

Use "physical screen" functions to retrieve real number of pixels (say, 2880x1800), and "virtual screen" functions to retrieve non-DPI resolution (say, 1440x900)
