739 /**
740  * Sets up a filter to process all events before they change internal state and
741  * are posted to the internal event queue.
742  *
743  * The filter is prototyped as:
744  * \code
745  * int SDL_EventFilter(void *userdata, SDL_Event * event);
746  * \endcode
747  *
748  * If the filter returns 1, then the event will be added to the internal queue.
749  * If it returns 0, then the event will be dropped from the queue, but the
750  * internal state will still be updated. This allows selective filtering of
751  * dynamically arriving events.
752  *
753  * \warning Be very careful of what you do in the event filter function, as
754  * it may run in a different thread!
755  *
756  * There is one caveat when dealing with the ::SDL_QuitEvent event type. The
757  * event filter is only called when the window manager desires to close the
758  * application window. If the event filter returns 1, then the window will
759  * be closed, otherwise the window will remain open if possible.
760  *
761  * If the quit event is generated by an interrupt signal, it will bypass the
762  * internal queue and be delivered to the application at the next event poll.
763  */
